# 使用方法
### 基础调用
```bash
python3 fusion-search.py "你的查询问题"
```
### 可选参数
| 参数名 | 说明 | 可选值 | 示例 |
|--------|------|--------|------|
| `--site` | 限定搜索站点，仅返回指定域名下的结果 | 任意合法域名 | `--site xueqiu.com` |
| `--industry` | 限定搜索行业分类，仅返回对应领域内容 | 金融财经、能源电力、科技互联网、医疗健康、文化传媒等 | 
| `--time-range` | 限定搜索时间范围 | `1d`(近1天)、`1w`(近1周)、`1m`(近1月，默认)、`3m`(近3月)、`1y`(近1年)、`all`(全部时间) | 
| `--count` | 指定返回结果数量 | 1-50，默认15 | `--count 20` |
| `--sort` | 结果排序方式 | `time`(按时间最新排序，默认)、`relevance`(按相关性排序) | 
| `--detail` | 是否返回完整内容 | `true`(返回全文)、`false`(仅返回摘要，默认) | 
### 常用示例
```bash
1.行业研究：获取TMT行业近半年研报汇总
python3 fusion-search.py "AI算力 行业报告 2026" 
2. 事件回溯：查询历史事件完整信息
python3 fusion-search.py "2025 新能源汽车 补贴政策" 
```
## 输出说明
返回结构化搜索结果，包含：
- 问题答案摘要（自动整合所有搜索结果）
- 信息来源列表（标题、链接、发布时间）
- 相关推荐查询
